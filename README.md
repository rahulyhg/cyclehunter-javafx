cyclehunter
===========

JavaFX application for graphically comparing historical stock and futures price data to hunt for cycles.


##### Dependencies: #####

- Java 1.8
- Maven 2


##### Running: ######

> git clone https://github.com/rluu/cyclehunter.git
>
> cd cyclehunter
>
> mvn compile
>
> mvn package
>
> ./run.sh
